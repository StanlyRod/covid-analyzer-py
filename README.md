# covid-analyzer-py
<h1>Personal project to analyze disaggregated data on covid statistics</h1>

<h1>How to use it ?</h1>

<h3>1</h3>
<br/>
Is recommended to create a python virtual environment, for that run the following command in the terminal:

python -m venv canalyzer
